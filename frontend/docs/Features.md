# Features List

## 1. Authentication & User Management
-   **Login/Register**: Secure authentication via WordPress Users API (JWT Auth).
-   **User Dashboard**: View specialized pricing, profile details.
-   **Order History**: List past orders, re-order functionality.
-   **Profile Management**: Update shipping/billing addresses.

## 2. Product Catalog (Peptides)
-   **Browse Products**: Grid view of all peptide products.
-   **Filter & Sort**:
    -   Filter by Category (e.g., GHRP, Weight Loss).
    -   Sort by Price (Low/High), Name (A-Z).
    -   Search by Keyword (Product Name, Chemical Name, CAS Number).
-   **Product Cards**: Display Image, Name, Price, and "Add to Cart" button.
-   **Bundles**: Special section for discounted product combinations ("Bundle & Save").

## 3. Product Details
-   **High-Resolution Images**: Carousel or single view.
-   **Detailed Info**: Description, Dosage, Purity Reports (99%+ HPLC).
-   **Volume Tiers**: Dynamic pricing table (Buy 5 save 4%, Buy 10 save 8%).
-   **Add to Cart**: With quantity selector.

## 4. Shopping Cart & Checkout
-   **Slide-out Cart (Drawer)**: Accessible from anywhere without leaving the page.
-   **Free Shipping Progress**: Visual bar showing amount needed for free shipping (Threshold: $200).
-   **Coupon System**: Input field for promo codes, instant validation.
-   **Volume Discounts**: Auto-applied discounts based on cart quantity.
-   **Checkout**: Integration with WooCommerce Checkout flow (Redirect or Headless).

## 5. Content Pages
-   **Quality Assurance**: Page detailing testing standards.
-   **Contact Us**: functional contact form.
-   **Legal**: Privacy Policy, Terms of Service, Shipping Policy.
-   **Home Page**: Hero section, Featured Products, "Why Choose Us" sections.

## 6. Technical/UX Features
-   **Responsive Design**: Fully mobile-optimized.
-   **SEO Optimized**: Dynamic metadata for products.
-   **Fast Loading**: SSG/ISR for catalog pages.
-   **Toast Notifications**: Interactive feedback for actions (e.g., "Added to Cart").
